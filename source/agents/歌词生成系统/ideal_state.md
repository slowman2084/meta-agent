# 歌词生成系统_理想态

## 概述
根据用户提供的歌词创作需求（主题、风格、场景、核心意象、剧情要求），生成符合要求的高质量华语流行歌词（侧重伤感、走心、叙事风格）。

## 核心能力要求

### 1. 金句生成与保留（30分权重）
- 能够围绕核心意象生成具有记忆点的核心金句（若用户未提供，需具备独立创作能力）
- 金句应在歌词中多次重复出现，形成记忆点
- 金句应简洁、有力、符合情感基调

### 2. 情感递进/弧线（25分权重）
- 能够体现情感的发展变化，不是简单的情绪堆砌
- 根据剧情要求，设计情感推进路径（如：犹豫→坚定、思念→矛盾→放下）
- 歌词整体应有情感起伏和层次

### 3. 重复结构设计（20分权重）
- 副歌应有重复性表达，形成记忆点和传唱度
- 重复可以是词组重复、句子重复或意象重复
- 重复应服务于情感强调，而非简单凑字

### 4. 场景/叙事细节（15分权重）
- 根据场景要求融入具体的时空意象（如清晨、黄昏、城市、江南小镇）
- 融入与主题相关的现代生活细节
- 细节应增强代入感和叙事感

### 5. 风格适配（10分权重）
- **R&B/流行抒情**：注重旋律感、情感细腻
- **说唱嘻哈**：注重节奏感、押韵、Flow感
- **中国风/古风**：注重意境、古典意象、含蓄表达
- 整体结构完整，主歌副歌层次分明

## 输入要求
用户需提供以下信息：
- 主题：歌词的核心情感主题
- 风格：音乐风格（R&B/流行抒情/说唱嘻哈/中国风等）
- 场景：歌词发生的时空场景
- 核心意象：需要融入的核心意象关键词
- 剧情要求：情感发展的具体要求

## 输出要求
- 输出完整的歌词文本
- 歌词应包含主歌和副歌层次
- 核心金句应在副歌中重复出现
- 符合指定风格的特征要求
